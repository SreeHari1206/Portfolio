MBD and FEA of Extendable Trailer

Project Overview

The objective of this project is to analyze the trailer under specified load and boundary conditions and perform multibody dynamics analysis to evaluate structural integrity and dynamic performance.

Software Used

  • HyperMesh
  • OptiStruct
  • HyperView
  • MotionView

Analyses Performed

  • Linear Static Analysis
  • Nonlinear Static Analysis
  • Modal Analysis
  • Multibody Dynamics Analysis

Deliverables

  • Stress Distribution
  • Deflection Results
  • Modal Frequencies
  • MBD Performance Evaluation
  • Design Recommendations
